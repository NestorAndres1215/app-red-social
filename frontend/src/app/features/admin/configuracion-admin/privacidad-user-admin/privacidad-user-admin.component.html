Cuenta Privada

Si tu cuenta es pública, cualquier persona —dentro o fuera de Social Like— podrá ver tu perfil y tus publicaciones, incluso quienes no tengan una cuenta en la plataforma.

Si tu cuenta es privada, solo los seguidores que apruebes podrán ver el contenido que compartas, como fotos o videos, así como tus listas de seguidores y seguidos. Algunos datos básicos de tu perfil, como tu foto y nombre de usuario, siempre serán visibles para todos.
